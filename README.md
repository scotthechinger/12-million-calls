# 12 Million Calls Initiative — Site Prototype

Prototype website for the 12 Million Calls Initiative, a project of the Policing Project at NYU School of Law in partnership with Zealous and the Joyce Foundation.

## Pages
- **Home** — Hero, intro, video teaser, misconceptions accordion, map, news, about
- **Response** — Pre-response diversion, 7-step immersive in-person response, alternate destinations, second response
- **Responders** — Searchable video archive
- **Evidence** — Key findings, data, national map, research database
- **About** — Project overview, Policing Project, approach, values
- **Get Involved** — Audience-segmented CTAs, contact form
